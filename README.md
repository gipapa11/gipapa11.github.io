# geraldipapa.github.io

Personal academic website of Gerald Ipapa, Postdoctoral Fellow at Georgetown University — live at [geraldipapa.github.io](https://geraldipapa.github.io).

Built on the [academicpages](https://github.com/academicpages/academicpages.github.io) Jekyll template (MIT License); deployed via GitHub Pages from the `master` branch.

## Updating content

- **About page:** `_pages/about.md`
- **Research:** one markdown file per paper in `_publications/`
- **CV:** `_pages/cv.md` (mirror of `files/Ipapa_CV.pdf`; source of truth is `job-market/cv/Ipapa_CV.qmd`)
- **Talks / Teaching:** `_talks/`, `_teaching/`
- **Paper PDFs:** `files/`
- **Site settings:** `_config.yml`
